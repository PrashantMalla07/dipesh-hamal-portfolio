# Dipesh Bahadur Hamal Portfolio (React)

Modern, responsive portfolio website for a Civil & Rural Engineer, built with React + Vite.

## Features

- Professional hero, about, skills, projects, experience, education, and contact sections
- Earthy engineering-inspired theme (light and dark mode)
- Smooth scrolling and reveal animations
- Project gallery with lightbox preview
- Nepal project map section
- Downloadable CV button
- SEO-friendly metadata

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Update Personal Details

Edit `contactInfo` and section content in:

- `src/App.jsx`

Static files used by the website are in:

- `public/assets/docs`
- `public/assets/images/credentials`
- `public/assets/images/projects`
